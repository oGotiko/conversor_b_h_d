# Conversor de Números (Binário, Decimal, Hexadecimal)

Este é um **programa em C** que permite converter números entre **Binário, Decimal e Hexadecimal**, com foco em **entender como os cálculos funcionam**, não apenas usar funções prontas.

## Funcionalidades

- Conversões **Binário → Decimal** e **Binário → Hexadecimal**  
- Conversões **Decimal → Binário** e **Decimal → Hexadecimal**  
- Conversões **Hexadecimal → Binário** e **Hexadecimal → Decimal**  
- Menu interativo para escolha do tipo de conversão  
- Validação de entrada para evitar números inválidos  
- Fácil de estudar e modificar para fins educativos
